Q3. Open the "tidy_data.pdf" document in the repo, which is a paper called Tidy Data by Hadley Wickham.

  1. Read the abstract. What is this paper about?
  2. Read the introduction. What is the "tidy data standard" intended to accomplish?
  3. Read the intro to section 2. What does this sentence mean: "Like families, tidy datasets are all alike but every messy dataset is messy in its own way." What does this sentence mean: "For a given dataset, it’s usually easy to figure out what are observations and what are variables, but it is surprisingly difficult to precisely define variables and observations in general."
  4. Read Section 2.2. How does Wickham define values, variables, and observations?
  5. How is "Tidy Data" defined in section 2.3?
  6. Read the intro to Section 3 and Section 3.1. What are the 5 most common problems with messy datasets? Why are the data in Table 4 messy? What is "melting" a dataset?
  7. Why, specifically, is table 11 messy but table 12 tidy and "molten"?
  8. Read Section 6. What is the "chicken-and-egg" problem with focusing on tidy data? What does Wickham hope happens in the future with further work on the subject of data wrangling?
  1. This paper is about how to formulate clean data sets, in which every variable is a column, every observation is a row, and every type of observational unit is a table. Data tidying allows for each to manipulate datasets, which can then be modeled and visualized .
  2. The tidy data standard was designed to facilitate the initial exploration and analysis of the data, and to simplify the development of data analysis tools that work well together.
  3. Sentence #1: This sentence means that tidy datasets all have the same structure with different numbers of each component specific to the dataset (columns, rows, tables); messy data can be messy for a variety of non-uniform reasons- missing values, formatting, etc. Sentence #2: This sentence emphasizes the idea that when column names may hold multiple meanings, it becomes harder to distinguish between observations and variables in a data set. Functional relationships are easier to describe between variables than between rows, and it is easier to make comparisons between groups of observations than between groups of columns.
  4. Values are what make up a dataset, usually numbers or strings. Values belong to a variable and an observation. A variable contains all values that measure the same underlying attribute across units. An observation contains all values measured on the same unit across attributes.
  5. Tidy data is defined as every variable forming a column, every observation, and each type of observational unit forming a table.
  6. Five most common problems:
    1. Column headers are values, not variable names.
    2. Multiple variables are stored in one column.
    3. Variables are stored in both rows and columns.
    4. Multiple types of observational units are stored in the same table.
    5. A single observational unit is stored in multiple tables.
      The data is messy because the column names are values, and multiple types of observational units are stored in the same table (religion and income). Melting a dataset means turning the columns into rows; melting is parameterized by a list of columns that are already variables (colvars).
  7. Table 11- lots of missing values, date is split into month and year, d1, d2, d3... should all be in one column as values Table 12- columns are not values, date is now a combined column with year and month. numbers are now aligned in a column corresponding to tmax and tmin. more compact, readable, and organized.
  8. The Chicken and Egg problem is that tidy data is only as useful as the tools that work with it, then tidy tools will be tied to tidy data. This means that independently changing data tools will not improve workflow. Wickham hopes that others will build on his framework to develop even better data storage strategies and better tools. He also hopes to use human factors to improve the cognitive side of data analysis and improve the ability to design appropriate tools.
